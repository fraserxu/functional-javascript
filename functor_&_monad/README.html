Functor & Monad

本章要介绍两个神秘的概念, 如果你没接触过 Haskell 或者 scala, 这有可能是你从没见过的神秘概念.

我会通过把这两个概念带入到 JavaScript 并且实现, 来解释为什么我们需要 Functor & Monad. 他们能给我们的编程带来哪些好处.
