Functional JavaScript

这是一本你可能2小时就能看完的小书，但是涵盖了基本所有函数式编程的内容，还包含了一些 ECMAScript 6 定义的函数式新特性, 如箭头函数, 模式匹配等等. 还会介绍函数式一些重要概念在 JavaScript是如何实现即应用, 以及如何以函数式的思想编写 JavaScript 代码.

如果你属于:

可能本书并不适合你, 请先移步Learn JavaScript, 但如果函数式对你来说:

那么这本书将会对你会有所帮助.

我选用的 JavaScript 函数式库是 Eweda(Ramda的更函数式的实现, Eweda 更函数式, 但是 Ramda 要更 pratical 一些)

Comment为什么不用 Underscore Short version: 用 underscore 体会不到函数式的精髓。Long version: 请移步第二章

Note 由于会介绍 ECMAScript 6 的新特性, 书中很多写法都是 ECMAScript 6 标准, 只能在实现这些 feature 的浏览器(如 Firefox, 请目前参照各浏览器的实现情况)里运行. 另外, 大多数的例子源码都会在文章里的 jsbin 链接里.
